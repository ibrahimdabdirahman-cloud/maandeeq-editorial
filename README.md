# Maandeeq Restaurant — Website (Editorial edition)

A single-page site for Maandeeq Restaurant, Woolwich. Bold bordered-editorial style.

## What's here
- `index.html` — the whole site (Home, Menu, Order, Reserve, Catering, About, Reviews, Contact, FAQ)
- `support.js` — runtime needed by `index.html` (keep it next to index.html)
- `assets/` — brand logo + food photography

## Deploy on GitHub Pages
1. Create a repo and upload **all files in this folder** (keep `support.js` and the `assets/` folder next to `index.html`).
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Branch: `main`, folder: `/ (root)`. Save.
4. Your site goes live at `https://<username>.github.io/<repo>/`.

All paths are relative, so it also works opened straight from the folder.
